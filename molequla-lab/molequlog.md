# Molequlog - Molequla Lab Observations

A living observation journal tracking the autonomous evolution of four Molequla organisms — earth, air, water, fire — running daily on GitHub Actions. As runs accumulate in `molequla-lab/runs/`, this file summarizes what's happening across them.

## How to read this

Daily run reports live in `runs/YYYY-MM-DD/report.md`. Each report contains per-element logs, growth metrics, and evolution snapshots from that day's run.

This file distills key observations, patterns, and milestones across runs — a human-readable layer on top of the raw data.
